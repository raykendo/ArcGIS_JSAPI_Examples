ArcGIS JavaScript API Example Archives

This site will server as an archive of simple maps created with the ArcGIS JavaScript API. The goal is to create simple web maps using as many previous versions as possible. For my purposes, I'll be using it to test apps and bookmarklets against a range of API versions.

Map examples will be separated by version number, and expanded upon by specific features.
