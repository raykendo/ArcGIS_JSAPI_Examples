ArcGIS JavaScript API Examples

This site will server as an archive of simple maps created with the ArcGIS JavaScript API. For my purposes, I'll be using it as a testing ground to investigate parts of the API. The goal is to create simple web maps using as many previous versions as possible.

Map examples will be separated by version number, and expanded upon by specific features.

v. 3.13

  • Basic Map: A simple world map loading an ArcGISTiledMapService and an ArcGISDynamicMapService.
  • Compact Map: Same map as above, using the compact build for mobile applications.
